📖 How to Use This Tool

The Case Converter transforms your text into different letter cases in real-time. No sign-up or account needed.

  • Enter Text: Type or paste text into the input area. All conversions update instantly.
  • UPPERCASE: Converts every letter to uppercase — e.g., "HELLO WORLD".
  • lowercase: Converts every letter to lowercase — e.g., "hello world".
  • Title Case: Capitalizes the first letter of every word — e.g., "Hello World".
  • Sentence case: Capitalizes only the first letter of each sentence — e.g., "Hello world. This is great."
  • aLtErNaTiNg CaSe: Alternates between lowercase and uppercase for each letter — e.g., "hElLo WoRlD".
  • Copy: Click the copy icon next to each result to copy that specific version.

❓ Frequently Asked Questions

What text cases can this tool convert?
This tool converts text to five formats: UPPERCASE, lowercase, Title Case, Sentence case, and aLtErNaTiNg CaSe. All conversions happen simultaneously in real-time.
How do I copy a specific converted result?
Each conversion result has its own Copy button. Click the copy icon next to the result you want, and it will be copied to your clipboard instantly.
What is Title Case vs Sentence case?
Title Case capitalizes the first letter of every word (e.g., "The Quick Brown Fox"). Sentence case only capitalizes the first letter of each sentence (e.g., "The quick brown fox.").
Does the tool work offline?
Yes! The Case Converter runs entirely in your browser with JavaScript. Once the page is loaded, it works without an internet connection.
